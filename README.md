# 👁️ VisionGameApp: AI-Powered Vision Testing Game 🎮

## 🚀 Overview
VisionGameApp is an Android-based **AI-powered vision testing application** that transforms traditional eye screening into an engaging and gamified experience. It helps users identify potential vision problems through interactive mini-games and provides intelligent analysis with actionable insights.

> ⚠️ This app is for **screening purposes only** and does not replace professional medical diagnosis.

---

## ✨ Key Features

### 🎮 Gamified Vision Tests
- Color Vision Test (Ishihara-style plates)
- Visual Acuity Test (distance & clarity)
- Reaction Time Test
- Depth Perception Test *(upcoming)*

---

### 🤖 AI-Based Analysis (NEW)
- Smart interpretation of test results
- Severity detection (Normal / Mild / High Risk)
- Personalized feedback based on performance
- Future-ready Gemini AI integration

---

### 📊 Progress Tracking Dashboard
- Test history with timestamps
- Filter by test type (Color / Acuity / General)
- Status tags (🟢 Normal | 🟡 Mild | 🔴 High Risk)
- Score trends and performance insights

---

### 🧠 Smart Recommendations
- Suggested next steps after each test
- Doctor recommendation:
  - 👨‍⚕️ Ophthalmologist (Eye Specialist)
  - 👓 Optometrist (Vision Testing)
- Lifestyle and eye care tips

---

### 🎯 Real-Time Feedback System
- Instant result calculation
- Mistake tracking
- Performance-based evaluation

- Clean Material UI design

## 🧪 How It Works

1. User selects a vision test  
2. Completes interactive game  
3. App evaluates performance  
4. AI generates insights & severity  
5. Recommendations are provided
